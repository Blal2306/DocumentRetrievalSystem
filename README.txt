*** DOCUMENT RETRIEVAL SYSTEM ***

GENERATE THE DICTIONARY OVER THE COLLECTION

1) CD into the DictionaryGenerator
2) Run "make"
3) java assignmentP2 [source collection of documents]

USING THE DOCUMENT RETRIEVAL SYSTEM

1) Copy the files generated by the DictionaryGenerator to the outer directory
2) CD into the outer directory
3) Run "make"
4) java assignmentP3 

* The program will continue until EXIT is entered by the user.
